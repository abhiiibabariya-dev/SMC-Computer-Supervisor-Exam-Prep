## PART 1: ENGLISH TYPING

### 1.1 QWERTY Keyboard Layout

```
` 1 2 3 4 5 6 7 8 9 0 - =
  Q W E R T Y U I O P [ ]
   A S D F G H J K L ; '
    Z X C V B N M , . /
```

### 1.2 Home Row Position (MOST IMPORTANT)

| Left Hand | Right Hand |
|-----------|------------|
| Pinky → A | Index → J |
| Ring → S | Middle → K |
| Middle → D | Ring → L |
| Index → F | Pinky → ; |
| **Thumb → Space Bar** | **Thumb → Space Bar** |

**F and J keys have small bumps** — use them to find home position without looking.

### 1.3 Finger-Key Assignment

| Finger | Left Hand Keys | Right Hand Keys |
|--------|---------------|-----------------|
| Pinky | Q, A, Z, 1 | P, ;, /, 0 |
| Ring | W, S, X, 2 | O, L, ., 9 |
| Middle | E, D, C, 3 | I, K, ,, 8 |
| Index | R, T, F, G, V, B, 4, 5 | Y, U, H, J, N, M, 6, 7 |

### 1.4 Practice Plan (30 Days)

| Week | Focus | Target Speed |
|------|-------|-------------|
| Week 1 | Home row (ASDF JKL;) only | 10 WPM |
| Week 2 | Top row (QWERTY) + Home row | 15 WPM |
| Week 3 | Bottom row (ZXCVB) + All rows | 20 WPM |
| Week 4 | Speed building + Accuracy | 25-30 WPM |

### 1.5 Practice Passages

**Passage 1 (Easy):**
```
The quick brown fox jumps over the lazy dog. This sentence contains every
letter of the English alphabet. Practice typing this sentence multiple times
to improve your speed and accuracy. Remember to keep your fingers on the
home row and use the correct finger for each key.
```

**Passage 2 (Medium):**
```
Surat is the eighth largest city in India and the second largest city in
Gujarat. It is known as the Diamond City of India because about 90 percent
of the world's diamonds are cut and polished here. The city is situated on

```

**Passage 3 (Office-related):**
```

A government clerk must be proficient in both English and Gujarati typing.
Computer skills including knowledge of Microsoft Office are essential for

```

### 1.6 Speed Building Tips

1. **Never look at the keyboard** — train muscle memory
2. **Use all 10 fingers** — don't type with 2 fingers
3. **Accuracy first, then speed** — errors slow you down more
4. **Practice 30 minutes daily** — consistency beats long sessions
5. **Use online typing tools** — TypingClub, Keybr.com, 10FastFingers

## PART 2: GUJARATI TYPING (InScript Layout)

### 2.1 InScript Keyboard Layout

InScript = **In**dian **Script** — Standard Government of India keyboard layout.

**Key Principle:**
- **Left side of keyboard** = Consonants (વ્યંજન)
- **Right side of keyboard** = Vowels (સ્વર) and Matras

### 2.2 Important InScript Key Mappings

#### Vowels (Right Side)
| Key | Normal | With Shift |
|-----|--------|-----------|
| D | ् (halant) | अ (અ) |
| E | ा (આ matra) | आ (આ) |
| F | ि (ઇ matra) | इ (ઇ) |
| R | ी (ઈ matra) | ई (ઈ) |
| G | ु (ઉ matra) | उ (ઉ) |
| T | ू (ઊ matra) | ऊ (ઊ) |
| S | े (એ matra) | ए (એ) |
| W | ै (ઐ matra) | ऐ (ઐ) |
| A | ो (ઓ matra) | ओ (ઓ) |
| Q | ौ (ઔ matra) | औ (ઔ) |

#### Common Consonants (Left Side)
| Key | Gujarati Character |
|-----|-------------------|
| K | क (ક) |
| I | ग (ગ) |
| U | ह (હ) |
| H | प (પ) |
| J | र (ર) |
| L | द (દ) |
| ; | य (ય) |
| O | ध (ધ) |
| N | ल (લ) |
| V | न (ન) |
| B | व (વ) |
| M | स (સ) |

### 2.3 Half Letters (Conjuncts)

To type a half letter (જોડાક્ષર):
1. Type the first consonant
2. Press **D key** (halant ્)
3. Type the second consonant

Example: ક + ્ + ર = ક્ર

### 2.4 Practice Plan (30 Days)

| Week | Focus | Target |
|------|-------|--------|
| Week 1 | Learn vowels + common consonants | Familiarity |
| Week 2 | Simple words + short sentences | 8-10 WPM |
| Week 3 | Paragraphs + half letters | 15 WPM |
| Week 4 | Speed building + exam passages | 20-25 WPM |

## PART 3: EXAM DAY TIPS

### Before the Exam
1. **Know the software** — Usually Unicode/Shruti font based
2. **Check keyboard** — Test all keys before starting
3. **Adjust chair height** — Comfortable posture matters
4. **Calm down** — Don't panic, you just need to pass

### During the Exam
1. **Read the passage once** before starting
2. **Type at steady pace** — don't rush
3. **Accuracy > Speed** — wrong words count against you
4. **Don't waste time on difficult words** — move on, come back later
5. **Use backspace wisely** — fix errors quickly
6. **Keep eyes on screen** — not on keyboard

### Speed Calculation
```
WPM = (Total Characters Typed ÷ 5) ÷ Time in Minutes

Example: If you type 1250 characters in 10 minutes:
WPM = (1250 ÷ 5) ÷ 10 = 250 ÷ 10 = 25 WPM ✅
```

### Common Mistakes to Avoid
1. Looking at keyboard while typing
2. Using only 2-3 fingers instead of all 10
3. Typing too fast and making errors
4. Not practicing with actual exam-type passages
5. Not learning InScript layout (using Phonetic instead)
6. Panicking during the exam
