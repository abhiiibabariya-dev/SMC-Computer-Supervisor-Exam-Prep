### Quick Reference: Important Gujarati Grammar Rules

| Topic | Key Points |
|-------|------------|
| સ્વર સંધિ | અ+આ = આ, ઈ+ઈ = ઈ, ઉ+ઉ = ઊ |
| ગુણ સંધિ | અ+ઈ = એ, અ+ઉ = ઓ |
| વૃદ્ધિ સંધિ | અ+એ = ઐ, અ+ઓ = ઔ |
| તત્પુરુષ | બીજો પદ પ્રધાન |
| કર્મધારય | વિશેષણ + વિશેષ્ય |
| દ્વન્દ્વ | બંને પદ સમાન |
| બહુવ્રીહિ | બંને પદ ગૌણ, ત્રીજા અર્થમાં |
| દ્વિગુ | પ્રથમ પદ સંખ્યાવાચક |
