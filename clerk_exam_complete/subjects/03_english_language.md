### Quick Reference: Common Confusing Words

| Word Pair | Meaning |
|-----------|---------|
| Accept / Except | Accept = receive; Except = excluding |
| Affect / Effect | Affect = verb (influence); Effect = noun (result) |
| Advice / Advise | Advice = noun; Advise = verb |
| Principal / Principle | Principal = head/main; Principle = rule |
| Stationary / Stationery | Stationary = not moving; Stationery = paper/pens |
| Their / There / They're | Their = possession; There = place; They're = they are |
| Lose / Loose | Lose = not win; Loose = not tight |
| Its / It's | Its = belonging to it; It's = it is |
| Complement / Compliment | Complement = complete; Compliment = praise |
| Emigrate / Immigrate | Emigrate = leave country; Immigrate = enter country |
