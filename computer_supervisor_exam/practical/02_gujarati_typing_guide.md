# PART 2: KEYBOARD LAYOUTS EXPLAINED

## 2.1 InScript Layout (Most Common in Govt Exams)

InScript = **In**dian **Script** - Standard keyboard layout by Government of India.

### How InScript Works:
- **Left side of keyboard** = Consonants (વ્યંજન)
- **Right side of keyboard** = Vowels (સ્વર) and Matras
- Uses **standard QWERTY physical keyboard** but keys map to Gujarati characters

### InScript Keyboard Map:


## 2.2 Shruti / Phonetic Layout (Easier for Beginners)

In phonetic layout, you type the **English sound** and it converts to Gujarati:
- Type "ka" → ક
- Type "kha" → ખ
- Type "ga" → ગ
- Type "cha" → ચ

**Note:** Check your exam notification for which layout they use. If not specified, prepare for InScript.

# PART 3: HOW TO SET UP GUJARATI TYPING ON YOUR COMPUTER

## 3.1 Windows 10/11 Setup

### Step 1: Add Gujarati Language
1. Press **Win + I** → Open Settings
2. Go to **Time & Language** → **Language & Region**
3. Click **Add a language**
4. Search for **"Gujarati"**
5. Select **ગુજરાતી** and click **Install**
6. Wait for download to complete

### Step 2: Select InScript Keyboard
1. Go to **Settings** → **Time & Language** → **Language**
2. Click on **ગુજરાતી** → **Options**
3. Under Keyboards, you should see **Gujarati InScript**
4. If not, click **Add a keyboard** → Select **Gujarati InScript**

### Step 3: Switch Between English & Gujarati
- Press **Win + Space** to switch between languages
- OR press **Alt + Shift** to switch
- The language indicator appears in the taskbar (bottom right)
- **EN** = English | **ગુ** = Gujarati

### Step 4: Use On-Screen Keyboard (for learning)
1. Press **Win + Ctrl + O** to open On-Screen Keyboard
2. Switch to Gujarati input
3. The on-screen keyboard will show Gujarati characters
4. Use this to learn key positions initially

---

## 3.2 Using Google Input Tools (Alternative / Easier)

1. Go to: **google.com/inputtools/try**
2. Select **Gujarati** from the language dropdown
3. Type in English (phonetically) and it converts to Gujarati
4. Good for practice but **NOT the exam format** (exam uses InScript)

# PART 5: SPECIAL CHARACTERS & CONJUNCTS (જોડાક્ષર)

## 5.1 How to Type Conjuncts (Half Letters)

To type a conjunct (two consonants joined together), use the **Halant (્)** key:
- Halant key = **d** on keyboard

**Examples:**
```
ક્ક = k + d + k     (pakka - પક્કા)
ત્ર = l + d + j      (tra)
શ્ર = Shift+M + d + j  (shra)
ક્ષ = k + d + Shift+<  (ksha)
જ્ઞ = p + d + Shift+>  (gna)
સ્ત = m + d + l      (sta)
પ્ર = h + d + j      (pra)
```

# PART 7: EXAM DAY TIPS

## Before the Exam
1. **Know your layout** - Confirm if exam uses InScript or Phonetic
2. **Practice on similar computer** - Government offices usually have Desktop PCs
3. **Know the switch shortcut** - Win+Space or Alt+Shift to switch language
4. **Carry nothing** - No cheat sheets allowed

## During the Exam
1. **Read the passage FIRST** - Spend 30 seconds reading before typing
2. **Start SLOW** - Begin carefully, speed up gradually
3. **Don't panic on mistakes** - Use Backspace calmly
4. **Focus on accuracy** - 1 wrong word costs more than 1 missed word
5. **Don't look at keyboard** - If possible, maintain eye contact with screen
6. **Type continuously** - Don't stop to think, maintain rhythm
7. **Use both hands** - Proper hand placement saves time

## Speed Calculation
```
Formula: (Total correct characters ÷ 5) ÷ Time in minutes = WPM

Example:
- You typed 1250 characters correctly in 10 minutes
- Words = 1250 ÷ 5 = 250
- WPM = 250 ÷ 10 = 25 WPM ✅ PASS!

To get 25 WPM in 10 minutes:
- You need: 25 × 10 × 5 = 1250 correct characters
- That's about 125 characters per minute
- That's about 2 characters per second
- Very achievable with practice!
```

## Common Mistakes in Exam
| Mistake | Solution |
|---------|----------|
| Can't find a character | Use on-screen keyboard if allowed |
| Wrong keyboard layout | Check taskbar for ગુ indicator |
| Typing speed too slow | Focus on common words, skip difficult ones |
| Computer freezes | Raise hand, ask for help immediately |
| Forgot conjunct typing | Use halant (d key) between consonants |
| Mixed English characters | Make sure language is switched to Gujarati |
