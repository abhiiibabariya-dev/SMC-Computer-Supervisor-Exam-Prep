# EXAM OVERVIEW

| Detail | Info |
|--------|------|
| **Post** | Laboratory Technician - Post No. 33 |
| **Organization** | Surat Municipal Corporation (SMC) |

| **Qualification** | B.Sc (Chemistry/Microbiology) + DMLT |
| **Exam Type** | Written (OMR Based) - 100 MCQs |
| **Total Marks** | 100 |
| **Negative Marking** | -0.25 per wrong answer |
| **Duration** | 120 Minutes |
| **Medium** | Gujarati & English |
| **Age Limit** | 18-35 years (relaxation for reserved) |
| **Salary** | Rs. 29,200 - 92,300 (Level 5) |
| **Training Period Salary** | Rs. 17,500/month (first 3 years) |

### Eligibility
- B.Sc with Chemistry OR Microbiology as subject
- DMLT (Diploma in Medical Laboratory Technology) OR Lab Technician Course
- CCC/CCC+ Computer Certificate
- Age: 18-35 (General), +5 SC/ST/SEBC Males, +10 SC/ST/SEBC Females
