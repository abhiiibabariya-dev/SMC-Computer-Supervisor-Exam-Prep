## EXAM STRATEGY & 30-DAY PLAN

### Time Management Tips
- **Total Time:** 60-90 minutes for approximately 100 questions
- **Per Question:** Spend no more than 45-60 seconds per question
- **First Pass:** Attempt all easy questions first (ones you know immediately)
- **Second Pass:** Return to moderate difficulty questions
- **Third Pass:** Attempt difficult questions if time permits
- **Never leave any question unanswered** if there is no negative marking

### Section Attempt Order (Recommended)
1. **Start with your strongest subject** - Build confidence and save time
2. **General Knowledge** - Usually factual recall, quick to answer
3. **Computer Knowledge** - Straightforward, less calculation
4. **Gujarati / English Language** - Moderate time needed
5. **Mathematics** - Most time-consuming, attempt last
6. **Reasoning** - Can be tricky, requires focused thinking

### Negative Marking Strategy
- If there IS negative marking (typically 0.25 marks deducted per wrong answer):
  - Only attempt questions you are at least 50% sure about
  - Eliminate 2 options first, then guess from remaining 2
  - Never blindly guess
- If there is NO negative marking:
  - Attempt ALL questions - no question should be left blank
  - For unsure questions, eliminate obviously wrong options first

### Key Formulas to Remember
- **Simple Interest:** SI = (P x R x T) / 100
- **Compound Interest:** A = P(1 + R/100)^T
- **Profit %:** (Profit / CP) x 100
- **Loss %:** (Loss / CP) x 100
- **Average:** Sum of observations / Number of observations
- **Speed = Distance / Time**
- **Work:** If A does work in 'a' days, A's 1 day work = 1/a
- **Area of Circle:** pi x r^2
- **Circumference:** 2 x pi x r
- **Area of Triangle:** 1/2 x base x height
- **Volume of Cube:** side^3
- **Volume of Cylinder:** pi x r^2 x h

### Last-Minute Tips
1. Read each question carefully - many errors happen due to misreading
2. Watch out for words like "NOT," "EXCEPT," "FALSE" in questions
3. In Mathematics, always check units (km/m, hours/minutes)
4. In English, read all options before marking the answer
5. In Reasoning, draw diagrams for direction and arrangement questions
6. For Gujarati, pay attention to the exact spelling (jodni)
7. Carry all required documents: admit card, ID proof, stationery
8. Reach the exam center at least 30 minutes early
9. Stay calm - stress reduces performance
