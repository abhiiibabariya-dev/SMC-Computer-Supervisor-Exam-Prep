### 1.1 MS WORD - Expected Tasks

#### Task 1: Document Formatting
You may be asked to:
- Create a formal letter with proper formatting
- Set margins (Top: 1", Bottom: 1", Left: 1.25", Right: 1")
- Use specific font (Times New Roman 12pt or Arial 11pt)
- Apply bold, italic, underline to specific text
- Set line spacing (1.5 or double)
- Insert header and footer
- Add page numbers

#### Important Shortcuts for Word:
| Shortcut | Action |
|----------|--------|
| Ctrl+B | Bold |
| Ctrl+I | Italic |
| Ctrl+U | Underline |
| Ctrl+E | Center align |
| Ctrl+L | Left align |
| Ctrl+R | Right align |
| Ctrl+J | Justify |
| Ctrl+1 | Single spacing |
| Ctrl+2 | Double spacing |
| Ctrl+5 | 1.5 spacing |
| Ctrl+Enter | Page break |
| Ctrl+Shift+> | Increase font size |
| Ctrl+Shift+< | Decrease font size |

#### Task 2: Table Creation
- Create table with specified rows/columns
- Merge cells
- Apply borders and shading
- Auto-fit content
- Sort data in table

#### Task 3: Mail Merge (IMPORTANT!)
Steps:
1. **Start Mail Merge** → Letters
2. **Select Recipients** → Type new list / Use existing
3. **Insert Merge Fields** → Name, Address, etc.
4. **Preview Results**
5. **Finish & Merge** → Print/Edit individual

#### Task 4: Headers, Footers, Page Numbers
- Insert → Header → Choose design
- Insert → Footer → Choose design
- Insert → Page Number → Position

## PART 3: ENGLISH TYPING - SPEED BUILDING

### Typing Test Expected Format:
- Duration: 10 minutes
- Required speed: 30 WPM (Clerk) / 25 WPM (Computer Supervisor)
- A passage will be displayed, you type it
- Errors reduce your net speed

### Practice Passages (Government/Official Style):

**Passage 1:**
```
The Surat Municipal Corporation is committed to providing quality services
to all citizens of Surat city. As part of the Smart City initiative,
various digital services have been introduced to improve efficiency and
transparency in governance. Citizens can now access multiple services
through the online portal including birth and death certificates, property
tax payment, water bill payment, and building permission applications.
The corporation has also implemented an Intelligent Traffic Management
System to reduce congestion and improve road safety across the city.
```

**Passage 2:**
```
The Government of Gujarat has launched several initiatives to promote
digital literacy and e-governance across the state. The Digital Gujarat
portal provides a single window for citizens to access various government
services online. The Common Service Centers established in rural areas
ensure that even remote villages have access to digital services. Under
the Pradhan Mantri Gramin Digital Saksharta Abhiyan, millions of citizens
have been trained in basic digital skills including using computers,
accessing the internet, and making digital payments through UPI and
other platforms. These initiatives are transforming the way citizens
interact with the government and are making governance more efficient
and transparent.
```

**Passage 3:**
```
The diamond cutting and polishing industry of Surat is one of the largest
in the world. Approximately ninety percent of the world's diamonds pass
through Surat before reaching the international market. The industry
employs over eight lakh workers and contributes significantly to the
economy of Gujarat and India. The recently inaugurated Surat Diamond
Bourse is the world's largest office building, spread over sixty seven
acres at Khajod. This state of the art facility houses over four
```

## PART 4: EXAM DAY CHECKLIST

### Before Exam:
- [ ] Carry admit card + ID proof
- [ ] Reach 30 minutes early
- [ ] Know which software version (MS Office 2010/2013/2016)
- [ ] Know InScript keyboard layout
- [ ] Carry a pen for rough work

### During Exam:
- [ ] Read all instructions carefully
- [ ] Check keyboard functionality
- [ ] Adjust screen brightness and chair
- [ ] Save work frequently (Ctrl+S)
- [ ] Don't panic — it's qualifying, not merit-based
- [ ] Focus on accuracy over speed

### Common Mistakes to Avoid:
1. Not saving the file in correct format
2. Not using InScript layout for Gujarati
3. Forgetting to add file name as instructed
4. Not formatting as per instructions
5. Spending too much time on one task
6. Not reviewing before submitting
