## 6. KEY FINDINGS FROM SMC WEBSITE EXPLORATION

### What's on SMC Website:
1. **Current recruitment notification** (PRO.739 - 2295 posts)
2. **Answer keys** for 2023 exams (Computer Supervisor, Asst. Engineer)
3. **Exam results** for various posts
4. **Online application portal** for current recruitment
5. **Call letter downloads** (with seat number + mobile verification)

### What's NOT on SMC Website:
1. Old question papers (not published officially)
2. Detailed syllabus PDFs (only in notification)
3. Previous year cutoffs (not published)
4. Sample papers (not provided)

### Where to Find Old Papers:
- **MaruGujarat.net** — Best source for Gujarat govt exam papers
