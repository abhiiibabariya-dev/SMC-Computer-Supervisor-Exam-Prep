# 7. SOP: HOW TO FILL THE APPLICATION FORM (Step-by-Step)

## Step 1: Go to Official Website
- Open browser on your phone/computer
- Go to: **[suratmunicipal.gov.in/recruitment](https://www.suratmunicipal.gov.in/recruitment/)**
- Or search "SMC Recruitment 2026" on Google

## Step 2: Click "Apply Online"
- Find the notification for **"SMC Recruitment 2026"**
- Click the **"Apply Online"** button
- You will be redirected to the application portal

## Step 3: New Registration
- Click **"New Registration"** or **"Register"**
- Enter your details:
  - Full Name (as per SSC certificate)
  - Email ID (use your own, not shared)
  - Mobile Number (for OTP verification)
  - Create Password (strong, remember it!)
- You'll receive an **OTP on mobile** — enter it to verify
- Note down your **Registration Number** — SAVE IT!

## Step 4: Login to Portal
- Go back to the login page
- Enter your Registration Number / Email
- Enter Password
- Click **Login**

## Step 5: Select Post
- You'll see a list of available posts
- Select your post:
  - **Clerk Grade-III (Code-502)** for Clerk exam
  - **Computer Supervisor (Code-513)** for Computer Supervisor exam
- Click **"Apply"** next to your chosen post

## Step 6: Fill Personal Details
Fill in carefully (NO mistakes!):
- Full Name (English + Gujarati)
- Father's / Husband's Name
- Date of Birth
- Gender
- Category (General / OBC / SC / ST / EWS)
- PwD status (Yes/No)
- Marital Status
- Address (Permanent + Correspondence)
- Pin Code
- District
- State

> **IMPORTANT:** Your name must EXACTLY match your SSC certificate!

## Step 7: Fill Educational Details
- SSC (Board, Year, Percentage)
- HSC (Board, Year, Percentage, Stream)
- Graduation (University, Year, Percentage, Degree)
- Post-Graduation (if any)
- CCC Certificate Number & Date
- Any additional qualifications

## Step 8: Upload Documents
Upload the following in prescribed format:

| Document | Format | Size |
|----------|--------|------|
| Photo | JPEG/JPG | 20-100 KB |
| Signature | JPEG/JPG | 10-60 KB |
| Other certificates | PDF (if asked) | As specified |

### Photo Requirements:
- Recent passport-size photo
- White background
- Face clearly visible
- No sunglasses or cap

### Signature Requirements:
- Sign on white paper with black pen
- Scan or photograph it
- Crop to signature only

## Step 9: Preview & Verify
- **CAREFULLY** review all information
- Check name, DOB, category, education — ALL fields
- If anything is wrong, go back and correct it
- Once submitted, you **CANNOT change** most details

## Step 10: Pay Application Fee
- Select payment method:
  - Debit Card
  - Credit Card
  - Net Banking
  - UPI
- Enter payment details
- Complete the payment
- **SAVE the payment receipt / transaction ID**

## Step 11: Submit Application
- After payment, click **"Final Submit"**
- Your application is now submitted!
- **Download the Application Form PDF** — SAVE it!
- **Print 2 copies** for your records

## Step 12: Note Important Details
Save these in your phone:
- Registration Number
- Application Number
- Transaction ID / Payment Receipt
- Username & Password

## 8. COMMON MISTAKES TO AVOID

| Mistake | Solution |
|---------|----------|
| Name mismatch with SSC | Use EXACT name from SSC certificate |
| Wrong DOB | Double-check with your certificate |
| Wrong category selected | Confirm your category certificate |
| Photo too large | Compress to under 100KB |
| Waiting till last day | Apply in first 2-3 days |
| Not saving application | Download PDF immediately |
| Wrong mobile number | Use your OWN active number |
| Weak password | Use mix of letters + numbers + symbols |
| Not reading notification | Read FULL notification before applying |
| Multiple applications | Only ONE application per post |
