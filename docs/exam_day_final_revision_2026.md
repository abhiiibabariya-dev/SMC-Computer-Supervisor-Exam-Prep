## ⏳ 7-Day Final Revision Plan

Use this in the last week before your exam (start it the moment the exam date is announced).

| Day | Focus | What to do |
|-----|-------|-----------|
| **Day 7** | Computer Technical (high weightage) | OS, DBMS, Networks — revise notes + 1 mock test |
| **Day 6** | Computer Technical | Data Structures, Boolean Algebra, Programming basics |
| **Day 5** | Gujarat GK + Current Affairs | Districts, rivers, schemes, 2025–26 affairs |
| **Day 4** | Gujarati + English language | Grammar rules, common errors, vocabulary |
| **Day 3** | Full mock test (timed) | Simulate real exam; analyse every wrong answer |

| **Day 1** | Light revision + rest | Formula sheets only. Sleep early. No new topics. |

> **Golden rule:** In the last 48 hours, **revise — don't learn new things.** New topics raise anxiety and rarely add marks.

## 🔥 High-Weightage Topics (revise these first)

**Computer Technical (75 marks — the deciding section):**
1. Operating System — process, memory, scheduling, file systems
2. DBMS — normalization, SQL, keys, ACID properties
3. Computer Networks — OSI/TCP-IP layers, IP addressing, devices
4. Data Structures — arrays, stacks, queues, trees, sorting/searching
5. MS Office — shortcuts, formulas, features (very common)
6. Number systems & Boolean Algebra — conversions, logic gates
7. Cyber Security basics — malware, firewall, encryption, safe practices

**General Knowledge (10) + Current Affairs (5):**
- Gujarat: districts, rivers, dams, festivals, famous personalities
- Surat: history, Tapi river, textile & diamond industry, SMC functions
- National schemes, sports, awards, science (2025–26)

**Language (10):** Gujarati grammar + English grammar/vocabulary.

## 📝 OMR & Time-Management Strategy

- **Total: 100 MCQs, 100 marks.** Plan roughly **45–50 seconds per question.**
- **First pass:** Answer everything you know instantly (≈60 questions). Don't get stuck.
- **Second pass:** Return to the medium-difficulty ones.
- **Third pass:** Attempt remaining + educated guesses.
- **Negative marking?** Computer Supervisor has **NO negative marking** — so **attempt all 100**. (Clerk & some posts have −0.25 — for those, skip pure blind guesses.)
- **Fill the OMR bubble fully** with the correct pen. Don't overwrite. Recheck roll number & booklet series.
- **Mark answers in batches** (every 5 questions) — saves time vs. one-by-one.

## 💪 Mindset for Exam Day

1. **Sleep well** the night before — no late-night cramming.
2. **Eat light** before the exam — heavy food makes you drowsy.
3. **Stay calm.** If a question is hard, move on — come back later.
4. **Read each question fully** before looking at the options.
5. **Trust your preparation.** A confident mind scores higher than a panicked one.
